Business Entity Matching System

ML pipeline using RapidFuzz + XGBoost to improve duplicate detection accuracy by 35%.

Financial institutions often maintain business records across multiple platforms, legacy systems, and third‑party data sources. Over time, this leads to duplicate entities, inconsistent naming conventions, and fragmented profiles for the same business. These inconsistencies create operational risk, affect compliance workflows, and require large amounts of manual review. Traditional rule‑based matching approaches struggled with edge cases, spelling variations, and incomplete data, making accurate entity resolution a significant challenge

To address this, I designed and implemented a machine‑learning–driven Business Entity Matching & Scoring System. The solution combines RapidFuzz for fuzzy similarity scoring with an XGBoost‑based classification model to determine whether two business records represent the same entity. I built a complete ML pipeline that handled data preprocessing, feature engineering, threshold tuning, and automated retraining. The system evaluates multiple attributes — such as business names, addresses, registration IDs, and metadata — and produces a confidence score that drives downstream decision‑making.

By integrating fuzzy matching with supervised learning, the system achieved a 35% improvement in duplicate detection accuracy, significantly reducing manual reconciliation effort. It also introduced consistent scoring logic, automated threshold management, and scalable retraining capabilities. This project strengthened data governance, improved operational efficiency, and demonstrated how ML can modernize entity resolution in financial datasets

COVID‑19 X‑ray Classification

Ensemble deep learning model using ResNet50, InceptionV3, EfficientNetB0.

During the early stages of the pandemic, radiologists faced overwhelming workloads and inconsistent access to rapid diagnostic tools. Manual interpretation of chest X‑rays was time‑consuming and prone to variability, especially when distinguishing COVID‑19 from other respiratory conditions. To address this challenge, I developed an ensemble deep‑learning model capable of classifying chest X‑ray images with high accuracy and reliability.

The system combines ResNet50, InceptionV3, and EfficientNetB0, leveraging transfer learning to extract robust features from limited medical datasets. I applied extensive data augmentation to improve generalization and reduce overfitting, and used stratified evaluation metrics to ensure balanced performance across classes. The ensemble approach aggregates predictions from all three models, improving stability and reducing false positives.

This project demonstrates how AI can support clinical workflows by providing fast, consistent, and scalable image‑based screening. While not a replacement for medical diagnosis, the model highlights the potential of deep learning to assist healthcare professionals during high‑demand scenarios and improve early detection accuracy.

Full‑stack app using Django, PostgreSQL, MongoDB, React.

A university’s TA assignment process was heavily manual, involving spreadsheets, emails, and repeated back‑and‑forth between faculty and administrators. This led to scheduling conflicts, inconsistent workload distribution, and significant time spent coordinating assignments each semester.

Also implemented automated assignment logic, conflict detection, and real‑time updates to ensure accurate scheduling and transparency.

This system reduced manual coordination by 40%, improved scheduling accuracy, and provided a centralized, scalable solution for departmental operations.
